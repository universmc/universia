Voici un exemple de page HTML présentant une workMission : ```html Work Mission

Work Mission

Description de la mission

Créer une série de vidéos TikTok de 68 secondes chacune, présentant un personnage utilisant des lunettes AR et un drone virtuel. Le script comporte cinq scènes et inclut des fichiers multimédias associés.

Utilisation de GitHub

Nous utilisons GitHub pour gérer notre projet en suivant un système de branches de version pour chaque version de la vidéo. Ceci simplifie la collaboration et permet d'apporter des modifications sur plusieurs branches de la vidéo.

Script des vidéos

Le script du projet est accessible depuis le fichier 'script.json'. Il présente une description détaillée de chaque scène et renvoie vers les fichiers multimédias correspondants.

Fichiers multimédias

Chaque scène dispose de fichiers multimédias hébergés sur le référentiel GitHub, composés de fichiers audio (MP3), vidéo (MP4) et texte (TXT). Les noms et liens de ces fichiers peuvent être trouvés dans le fichier 'script.json'.

Scènes

  • Scène 1: Présentation du personnage et des lunettes AR
  • Scène 2: Activation des lunettes AR et apparition du drone virtuel
  • Scène 3: Contrôle du drone virtuel avec les lunettes AR pour accomplir une tâche
  • Scène 4: Utilisation du drone virtuel pour accomplir la tâche avec des effets visuels
  • Scène 5: Réussite de la tâche et message final
``` N'oubliez pas de créer un fichier CSS `styles.css` pour personnaliser l'apparence de votre page.
